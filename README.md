# Apache JMeter Performance Testing Demo

## Project Overview
This project demonstrates performance testing execution using Apache JMeter on the SauceDemo web application.

## Test Scenarios
- Homepage Navigation
- Login Workflow
- Product Page Validation

## Load Configuration
- 50 Virtual Users
- Ramp-Up: 10 Seconds
- Loop Count: 5

## Performance Results
- Total Samples: 750
- Error Rate: 0%
- Average Response Time: 26 ms
- Throughput: 74.9 requests/sec

## Tools Used
- Apache JMeter 5.6.3

## Artifacts Included
- JMeter Test Plan (.jmx)
- Aggregate Report
- Summary Report
- HTML Dashboard
- Performance Test PDF Report
